# Clasificarea poziției jucătorilor de fotbal după statistici reale

## Sursa datelor
- **Dataset:** All Football Players Stats in Top 5 Leagues 23/24
- **Link:** https://www.kaggle.com/datasets/orkunaktas/all-football-players-stats-in-top-5-leagues-2324
- Data descărcării: 24 martie 2026

## Obiectivul proiectului
Să clasifice poziția unui jucător de fotbal (atacant, mijlocaș, 
fundaș, portar) pe baza statisticilor sale reale din meciuri, 
precum goluri, assist-uri, pase reușite și deposedări, folosind 
modele de clasificare.

## Tehnologii utilizate
- Python 3
- O sa fie actualizat pe parcurs
